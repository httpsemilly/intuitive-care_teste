# Intuitive Care - Teste

Teste técnico para vaga de estágio em desenvolvimento na Intuitive Care.

## Sobre o Projeto

Este repositório contém a resolução dos 4 testes práticos propostos pela Intuitive Care, abordando:

- **Teste 1:** Integração com APIs públicas
- **Teste 2:** Transformação e validação de dados
- **Teste 3:** Banco de dados e análise SQL
- **Teste 4:** Desenvolvimento de API e interface web

## Tecnologias Utilizadas

- Python 3.8
- MySQL 8.0
- Flask
- Vue.js 3.5

## Documentação

Cada teste possui um arquivo README com as seguintes informações:

- Instruções de execução
- Decisões técnicas tomadas
- Trade-offs considerados
- Tratamento de inconsistências

## Como Executar

### Pré-requisitos

```bash
# Python 3.8 instalado
python --version

# MySQL 8.0 instalado e rodando
mysql --version
```

### Execução

1. Clone o repositório:

```bash
git clone https://github.com/httpsemilly/intuitive-care_teste

cd Teste_EmillyCavalcante
```

2. Crie um ambiente virtual

```bash
# Para Windows:

python -m venv venv
venv\Scripts\activate
```

3. Instale as dependências

```bash
pip install -r requirements.txt
```
